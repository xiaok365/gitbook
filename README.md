# Introduction

**专注计算机算法和数学等知识，以及一些生活中的应用场景，和大家一起用科学的眼光看世界**

1. 安装GitBook
  ```
  npm install gitbook-cli -g
  ```
2. 查看版本号
  ```
  gitbook -V
  ```
3. 安装插件
  ```
  gitbook install ./
  ```
4. 运行
  ```
  gitbook serve
  ```
5. 打包静态文件
  ```
  gitbook build
  ```
